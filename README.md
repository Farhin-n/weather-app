# Current Location Weather

A weather app that gives the current weather at the current location.
Uses Current Weather Data API from OpenWeatherMaps

# Getting Started
- Create an account with OpenWeatherMap. Go to your username and click on API keys. If a key doesn't exist then create one.
- Create a **.env** file in the main app folder and type the following
  ```
    REACT_APP_API_URL = 'https://api.openweathermap.org/data/2.5'
    REACT_APP_API_KEY = Paste your API key here.
    REACT_APP_ICON_URL = 'https://openweathermap.org/img/w'
  ```
- Clone the repository, install dependencies and run
  ```
  npm install
  npm start
  ```
